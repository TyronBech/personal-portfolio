
# Personal Portfolio

A minimal personal portfolio site built with Vite + React and Tailwind CSS.

## Overview

This repository contains a small, fast portfolio used to showcase projects and contact information. It uses a modern frontend toolchain for a lightweight development and production build process.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Project Structure (summary)

- `portfolio/` — main app folder
- `src/` — source files (components, assets, styles)
- `public/` — static assets
- `index.html`, `vite.config.js`, `tailwind.config.js`
